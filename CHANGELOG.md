# Change Log

All notable changes to the "Synthwave 80's" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [2.0.1] - 2026-02-04

### Fixed
- Improved comment contrast for better accessibility (WCAG compliance)
- Fixed deprecated `editorIndentGuide.background` keys
- Enabled bracket pair colorization across all themes
- Added semantic highlighting support

### Changed
- Updated VS Code engine requirement to ^1.80.0
- Removed incorrect `dependencies` field from package.json
- Improved `.vscodeignore` to reduce extension size

### Added
- Added `license`, `bugs`, and `homepage` fields to package.json
- Added semantic token colors for better syntax highlighting

## [2.0.0] - 2023-01-15

### Added
- New theme variant: "The Shredder" - a more muted, professional synthwave theme
- Comprehensive workbench color coverage (260+ colors)
- Full terminal ANSI color support
- Git decoration colors
- Diff editor colors

### Changed
- Completely redesigned Future Blue and Mutagen Ooze themes
- Improved syntax highlighting for major languages (JS/TS, Python, PHP, Ruby, Go, Rust)

## [1.0.0] - Initial Release

- Initial release of Synthwave 80's theme
- Future Blue variant
- Mutagen Ooze variant
